## Overview

Sales Analytics & Forecasting System is an end-to-end data analytics project built using Python, SQL, Excel, and Streamlit. The application enables users to upload sales data, analyze business performance, visualize key metrics, and generate sales forecasts through an interactive dashboard.

This project demonstrates practical skills in data analysis, data visualization, database management, and business intelligence.

---

## Features

### Data Management

* Upload sales data from CSV or Excel files
* Store and retrieve data using SQLite
* Automated data processing and validation

### Business Analytics

* Total Revenue Analysis
* Total Orders Tracking
* Average Order Value Calculation
* Product Performance Analysis
* Region-wise Revenue Analysis

### Data Visualization

* Interactive Revenue Trend Charts
* Product-wise Sales Comparison
* Regional Revenue Distribution
* Forecast Visualization

### Forecasting

* Revenue prediction for future periods
* Trend-based forecasting using Machine Learning
* Business performance insights

## Installation

Clone the repository:

```bash
git clone https://github.com/akshat5904/Sales-Analytics-Forecasting-System.git
```

Navigate to the project directory:

```bash
cd Sales-Analytics-Forecasting-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
